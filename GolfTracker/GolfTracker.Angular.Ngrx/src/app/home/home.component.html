The Golf Tracker application allows golfers to track their scores as they play golf.

Directions

  1. Golf Clubs - first enter the golf clubs, courses and tees.
  2. Manage Golfers - then enter a golfer
  3. Add Round - and finally add a round of golf for the golfer

The Architecture

  • UI - light-weight Angular 4 framework
  • Data - the application is flexible to handle different data stores. The Angular services point to RESTful endpoints for data. Currently it is setup to use either Azure DocumentDB or SQL Azure.

System Requirements

  • NodeJs and NPM
  • Visual Studio 2015 or later
  • .Net Core 1.1
  • Azure DocumentDB
  • SQL Azure
